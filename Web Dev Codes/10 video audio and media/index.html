We can have the thumbnail image for the video using the `poster` attribute, which in this case is the image itself.

Along with this we can use more attributes like `preload`, `loop`, and `muted`for videos.

Preload attribute specifies when the media should be loaded. for example, `preload="auto"` loads the video as soon as the page is loaded and `preload="none"` does not load the video until the user clicks on it whereas `preload="metadata"` loads only the metadata.

In audio we have similar attributes like `preload`, `loop`, and `muted`.
